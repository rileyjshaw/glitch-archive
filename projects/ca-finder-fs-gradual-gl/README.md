# WIP CA Finder

It would be cool to eventually [overkill it like this](https://github.com/piellardj/game-of-life-webgl,) but [starting simple is a good start](https://nullprogram.com/blog/2014/06/10/)

Remix: rules graaaadually evolve. Now with WebGL!

---

Add or remove options from the `states` variable if u wanna.

## Controls:

⬇️: Restart the current rule with a random grid.

➡️: Mutate the current rule a bit.

⬆️: Get a completely new rule.

⬅️: Undo your last change to the rule.
